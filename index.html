Visão Geral do LuusOS

O LuusOS é um projeto independente de desenvolvimento de sistemas operacionais (OSDev) focado em arquiteturas x86 e desenvolvimento de baixo nível. O principal objetivo é o entendimento e a implementação de estruturas de kernel fundamentais partindo do zero.

O ambiente operacional atual realiza o boot de modo protegido e interage diretamente com o hardware sem camadas intermediárias.

LuusOS Overview

LuusOS is an independent operating system development (OSDev) project focused on x86 architectures and low-level development. The main goal is to understand and implement fundamental kernel structures from scratch.

The current operating environment boots into protected mode and interacts directly with the hardware without any intermediary layers.

Arquitetura do Kernel

O design do LuusOS segue a filosofia de um kernel monolítico modularizado, dividindo-se entre o código de setup em Assembly e a lógica de controle principal escrita em C estruturado de alta performance.

Kernel Architecture

The LuusOS design follows the philosophy of a modularized monolithic kernel, split between Assembly setup code and core control logic written in highly performant structured C.

; Typical LuusOS boot structure
global _start
extern kernel_main

_start:
    cli                  ; Disable hardware interrupts
    mov esp, stack_space ; Set up stack pointer
    call kernel_main     ; Jump to C code
    hlt                  ; Halt CPU if kernel returns

Driver de Vídeo VGA (Modo Texto)

Abaixo está o modelo de escrita direta na memória de vídeo mapeada em 0xB8000. O buffer de texto aceita caracteres acompanhados de bytes de atributos (cores de fundo e primeiro plano).

VGA Video Driver (Text Mode)

Below is the standard direct write model into the video memory mapped at 0xB8000. The text buffer accepts characters paired with attribute bytes (background and foreground colors).

// Direct write driver into VGA video buffer
void vga_print_string(const char* str, char color) {
    volatile char* vga_buffer = (volatile char*)0xB8000;
    while (*str != 0) {
        *vga_buffer++ = *str++; // Send character
        *vga_buffer++ = color;  // Send color attribute byte
    }
}

Global Descriptor Table (GDT)

A GDT define as características dos segmentos de memória do sistema. O LuusOS configura os seletores de segmento de modo plano (Flat Model), segmentando código e dados do Kernel com privilégio máximo (Ring 0).

Global Descriptor Table (GDT)

The GDT defines the characteristics of the system's memory segments. LuusOS configures segment selectors using a Flat Model, partitioning Kernel code and data with maximum privilege (Ring 0).
